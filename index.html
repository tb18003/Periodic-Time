Periodic Time

Un programa en Java sobre la Tabla Periódica :D

Prueba también la nueva web hecha con Bootstrap

Este programa, combina el ser portable, rápido y ser gratis. Un resumen rápido de esta aplicación es tener mucha información sobre unos átomos invisibles. Perfecto cuando te quedas sin internet, y necesitas la tabla periódica, ya bien sea en clase o para formular en tu casa.

Otro detalle de esta aplicación a tener en cuenta, es que tiene un modo oscuro, para que afecte de una manera más leve a nuestra vista. Al usar en todas las IDE's este modo, decidí implementarlo en mi app también. Todos sabemos que el fondo blanco, a veces cansa.

También podrás contribuir con el programa, mediante el repositorio de Periodic Time alojado en GitHub, si te animas a mejorar esta aplicación, ve a mi perfil de GitHub, y ahí estará el código del programa, además, hay un botón dentro de la aplicación, para que haya un acceso más fácil :D.
